# Frontend Developer Test Case

Реализован интерфейс просмотра коротких видео с вертикальным листанием на vanilla JavaScript.

- `src/data.js` — источник данных.
- `src/utils.js` — утилиты форматирования.
- `src/feedView.js` — слой представления, отвечает только за рендер и доступ к DOM.
- `src/feedController.js` — слой логики: autoplay, observer, keyboard navigation, status updates.
- `src/main.js` — точка сборки зависимостей и инициализации.
